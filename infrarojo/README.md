# Sensores Infrarrojos IR

Lee el estado de los dos sensores IR y lo muestra por el Monitor Serie cada 500ms.
Útil para detectar línea negra en el suelo (sigue líneas).

## Qué aprendes
- digitalRead vs analogRead
- Sensores digitales: solo devuelven 0 o 1
- 0 = superficie clara, 1 = superficie oscura (o viceversa según sensibilidad)
- Ajuste de sensibilidad con potenciómetro

## Pines usados
- Pin 2 → sensor IR izquierdo
- Pin 3 → sensor IR derecho
