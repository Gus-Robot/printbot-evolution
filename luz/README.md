# Sensores de luz LDR

Lee la intensidad de luz de los dos sensores LDR y la muestra por el Monitor Serie cada 500ms.

## Qué aprendes
- Diferencia entre pines analógicos (A2, A3) y digitales
- analogRead vs digitalRead
- Valores analógicos entre 0 y 1023
- Valor bajo = mucha luz, valor alto = poca luz

## Pines usados
- Pin A2 → sensor LDR izquierdo
- Pin A3 → sensor LDR derecho
