# Huye Luz

El robot detecta de qué lado viene más luz y huye de ella usando los sensores LDR.

## Qué aprendes
- Lógica condicional if / else if / else
- Comparar valores de sensores para tomar decisiones
- Combinar sensores y actuadores en un mismo sketch

## Cómo funciona
- Más luz izquierda → gira derecha para huir
- Más luz derecha → gira izquierda para huir
- Misma luz → marcha atrás

## Pines usados
- Pin A2 → sensor LDR izquierdo
- Pin A3 → sensor LDR derecho
- Pin 8 → servo rueda izquierda
- Pin 9 → servo rueda derecha
