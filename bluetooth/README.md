# Control Bluetooth

Control manual completo del robot desde el móvil via Bluetooth.

## Qué aprendes
- Comunicación serie con Serial.available() y Serial.read()
- Variables de tipo char para leer caracteres
- Estructura if/else if para múltiples comandos
- Importancia de comprobar Serial.available() antes de leer

## Comandos
- `U` → adelante
- `D` → atrás
- `L` → izquierda
- `R` → derecha
- `S` → parar

## App recomendada
Arduino Bluetooth Controller — conectar a ZUM_BT328

## Pines usados
- Pin 8 → servo rueda izquierda
- Pin 9 → servo rueda derecha
- Baudrate BT: 19200
