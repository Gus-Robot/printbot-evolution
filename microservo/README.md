# Microservo cabeza

Mueve el microservo de la cabeza de lado a lado pasando por el centro.

## Qué aprendes
- Control de servo estándar (posición fija) vs servo de rotación continua
- write() con ángulos: 0°, 90°, 180°
- Diferencia entre servo de ruedas (rotación continua) y microservo de cabeza (posición)

## Secuencia de movimiento
0° → 90° → 180° → 90° → repite

## Pines usados
- Pin 11 → microservo cabeza
